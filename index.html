Venti

Arduino C++ Ajax

The Venti is an autonomous temperature control system whiches uses a ESP32 to serve an Ajax web server over the local network. The webserver utilises websockets to update users pages in real-time without refreshing. web interface is used in combination with a LoRa gateway to create an interface for users to send and receive vent data through the RFM95W LoRa transceiver. System also utilizes a SPIFFS library to create a solid state file storage system on the ESP32.
