Portfolio

SpotiRy

Node.js Javascript

Utilizes the Spotify API to generate a spotify wrapped based on lifetime listening statistics instead of yearly. Also has a built in queue generator that hand picks a selection of recommended songs based on your listening habits

Movie Nomination

React Javascript

Utilizes the OMDB movie database API to allow users to search for movies and rank them for an award cermony. React drag and drop components allowing users to adjust movie positions after selection.

Venti

Arduino C++ Ajax

The Venti is an autonomous temperature control system whiches uses a ESP32 to serve an Ajax web server over the local network. The webserver utilises websockets and a LoRa gateway to send and receive vent data through the RFM95W LoRa transceiver.

Amazon Cruncher

Node.js Javascript

Amazon Cruncher takes csv's and generates a detailed order history report to help users visualize and understand their spending patterns and gain deeper understanding of where their money is going.

Frontier

C# Unity Engine

This is a first person battle royal shooter game made in unity with multiplayer capability and multiple classes and guns to choose from. 3 teams all start on seperate corners of the map and try to lead a payload to the center, they can speed up the payload by collecting gold from the other teams

Self Watering Plant

Arduino Esp-32

A completely self sustaining plant pot that uses sensors to detect soil moisure levels and pulls water from a internal tank. This system uses a web based application as an interface to set watering schedules and moisture thresholds

HVAC System

Esp-32 Javascript

System use to remotely control home HVAC system from a web based application hosted entirely on the users home network, and acccessable through any device. Utilizes a SPIFFS library to create a solid state file storage system on the ESP32.

BattlePass Tracker

Node.js Javascript

Used to determine progress for different battlepasses, to help you determine if you are on track to finish the entire battlepass before a season is over. Also includes in game currency converters
